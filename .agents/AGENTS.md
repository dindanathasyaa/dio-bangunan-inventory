# Project-Scoped Rules for Dio Bangunan Inventory

## Auto-Commit and Push
Whenever you make any changes to the codebase, you MUST automatically run the following commands to push the changes to the user's GitHub repository:
1. `git add .`
2. `git commit -m "Your descriptive commit message"`
3. `git push`

Do this automatically without waiting for explicit permission from the user each time.
